# Installation on Microsoft Windows

1. Download the latest mzmine Windows installer (.msi)
2. Double-click the installer to run the installation
3. Windows might ask if you trust mzmine, select yes.
4. Run mzmine by double-clicking the mzmine.exe (can be found by pressing the Windows button and searching for mzmine)

## Updating mzmine
There is no need to deinstall mzmine, a new version can be installed over an older version. 
Only in rare cases mzmine might ask to remove an old version before installing a new version.

The mzmine configuration, presets, and other files are kept and shared between versions.
